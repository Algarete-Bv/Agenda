# Agenda Electrónica CRUD

Sistema de gestión de contactos desarrollado en C# mediante arquitectura en capas, implementando operaciones CRUD (Create, Read, Update and Delete) para la administración eficiente de una agenda electrónica.

## Descripción del Proyecto

La aplicación permite registrar, consultar, actualizar y eliminar contactos dentro de una agenda electrónica, manteniendo una estructura organizada y escalable gracias al uso de arquitectura en capas.

El proyecto fue desarrollado siguiendo buenas prácticas de separación de responsabilidades, distribuyendo la lógica del sistema en diferentes capas para facilitar el mantenimiento, reutilización y escalabilidad del código.

## Arquitectura Implementada

El sistema está estructurado bajo una arquitectura en capas compuesta por:

- **CapaPresentacion:** interfaz gráfica e interacción con el usuario.
- **CapaNegocio:** procesamiento de reglas de negocio y validaciones.
- **CapaDatos:** acceso y manipulación de datos.
- **CapaEntidad:** definición de entidades y modelos del sistema.

## Funcionalidades

- Registro de contactos
- Consulta de información
- Modificación de registros
- Eliminación de contactos
- Gestión organizada de datos
- Interfaz de escritorio amigable

## Tecnologías Utilizadas

- C#
- .NET Framework
- Windows Forms
- SQL Server
- ADO.NET
- Visual Studio 2022

## Objetivo

Aplicar los fundamentos de programación orientada a objetos y arquitectura en capas en el desarrollo de una aplicación de escritorio funcional para la gestión de información.
